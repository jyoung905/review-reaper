# Review Reaper Reply Handling SOP

## Current authority
Atlas can monitor and draft replies. External replies should not be sent without James approval unless James explicitly delegates sending authority.

## If a prospect replies “yes / send it over”
Goal: do not push Stripe yet. Deliver value first.

Draft response:

> Absolutely — I’ll put together a quick mini-audit for you.
>
> I’ll include:
> - the recent negative review themes I found,
> - 2–3 suggested response drafts,
> - and a short note on what I’d fix first from a reputation standpoint.
>
> I’ll send it over shortly.
>
> — James

Then create mini-audit within 24h.

## If a prospect asks “what is this / how does it work?”
Draft response:

> Fair question. Review Reaper is a simple review-response service for local businesses.
>
> I monitor public reviews, flag the negative ones, draft professional responses, and summarize the recurring complaint themes. Nothing gets posted automatically — you approve/edit anything before using it.
>
> I’m offering a free mini-audit first so you can see the quality before deciding if monthly monitoring is worth it.

## If a prospect asks price
Draft response:

> Ongoing monitoring is $97/month for one location. That includes negative review monitoring, response drafts, and a short complaint-theme summary.
>
> But I’d rather send the free mini-audit first so you can judge whether it’s useful.

## If a prospect is angry / says stop emailing
Draft response:

> Understood — sorry for the interruption. I won’t contact you again.

Then mark target `do_not_contact`.

## If a prospect wants to subscribe immediately
Do not just send them into a dead-end.

Draft response:

> Great — before you subscribe, I’ll confirm the setup so we can actually deliver the service properly.
>
> Please send:
> 1. business name/location,
> 2. main review profile link,
> 3. preferred response tone,
> 4. who should receive drafts for approval.
>
> Once I have that, I’ll send the subscription link and start the first review scan.

## Internal follow-up rule
If no reply after 5 business days:
- send one short follow-up,
- then stop unless they engage.

Follow-up draft:

> Quick follow-up — want me to send over the free review mini-audit for {business_name}?
>
> If not useful, no worries and I won’t follow up again.
